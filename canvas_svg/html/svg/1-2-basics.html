1-2 、 基本图形和属性

基本图形
<rect> 、<circle> 、<ellipse> 、<line> 、 <polyline> 、 <polygon>
基本图形

fill 、 stroke 、 stroke-width 、 transform 


            
    <rect>
        x : 起点x坐标
        y : 起点y坐标
        width : 宽
        height : 高
        rx : 圆角x
        ry : 圆角y
            
        

            
                <circle>
                    cx : 圆心x坐标
                    cy : 圆心y坐标
                    r : 半径
            
        

            
                <ellipse>
                    cx : 圆心x坐标
                    cy : 圆心y坐标
                    rx : x轴半径
                    ry : y轴半径
            
        

            
                <line>
                    x1 : 起点x坐标
                    y1 : 起点y坐标
                    x2 : 终点x坐标
                    y2 : 终点y坐标
            
        

            
                <polyline>
                    points ：x1 y1 x2 y2 ...
